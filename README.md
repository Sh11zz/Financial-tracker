# Financial-tracker
## Веб-приложение для учета личных финансов
>пользователь фиксирует доходы и расходы, управляет счетами и категориями, а система предоставляет аналитику и отчеты

## Стек
* Frontend: React
* Backend: Flask, Django
* Database: SQLite3
